# 📋 회원관리 및 질의응답 시스템 (Q&A Board)

## 🖥️ **프로젝트 소개**

이 프로젝트는 Python과 Flask를 기반으로 구축된 **회원관리 및 질의응답 시스템**입니다. 사용자는 회원가입 후 로그인하여 질문을 게시하고, 답변과 댓글을 통해 소통할 수 있는 기능을 제공합니다. 해당 시스템은 학습 목적으로 제작되었으며, 게시판 형태의 기본적인 웹 애플리케이션 구축 과정을 익히는 데 중점을 두었습니다.

---

## ⚙️ **주요 기능**

1. **회원관리**

   - 회원가입
   - 로그인 및 로그아웃
   - 비밀번호 변경

2. **질의응답 게시판**

   - 질문 작성, 수정, 삭제
   - 질문 목록 보기
   - 질문 상세 페이지
   - 답변 및 댓글 작성
   - 게시글 검색 및 정렬 기능
   - 추천 기능

3. **검색 및 필터링**

   - 질문 제목 및 내용으로 검색 가능
   - 최신순, 인기순 등 다양한 정렬 기준 제공

---

## 🛠️ **기술 스택**

- **Backend**: Python, Flask
- **Frontend**: HTML5, CSS3, JavaScript, jQuery, Bootstrap
- **Database**: PostgreSQL
- **Hosting**: AWS
- **Version Control**: Git
